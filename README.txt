MN Coaching Website
==================

How to open
-----------
1) Open the folder: mn-coaching
2) Double-click: index.html

Files
-----
- index.html  (page content)
- styles.css  (black/white styling)
- script.js   (mobile menu + contact form mailto)

Important
---------
In script.js there is a placeholder email address:
  hello@mncoaching.example
Replace it with your real email so the "Create enquiry email" button works.
